# Excel Sales and Store Data Analysis Project

## Project Overview

This project demonstrates the use of Microsoft Excel for data analysis, reporting, and business decision support.

It includes grocery sales analysis, store lookup using VLOOKUP, financial analysis using HLOOKUP, and student grading using IF functions.

## Skills Demonstrated

- Excel formulas and functions
- Data analysis and reporting
- VLOOKUP and HLOOKUP
- IF logical functions
- Data cleaning and organization
- Sales analysis

## Tools Used

- Microsoft Excel

## Key Features

- Calculated total sales value using formulas
- Performed store lookup using VLOOKUP
- Retrieved financial data using HLOOKUP
- Applied IF function to categorize performance
- Organized and analyzed structured datasets
